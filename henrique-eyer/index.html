CSS Box Model

O que vamos aprender?

Hoje você aprenderá a usar o Box Model do CSS, as propriedades fisicas dos elementos HTML e suas regras. Sendo mais simples, você aprenderá a modificar o design ou layout dos seus elementos HTML mudar seu tamanho, colocar margens ou bordas.

Você será capaz de:

  • Aprender a mexer com o layout dos elementos HTML e seu funcionamento.
  • Entender o funcionamento do Box Model do CSS.
  • Quais são suas propriedades e o que elas modificam.

O que é o Box Model do CSS?

São as propriedades que definem o layout e o design de um elemento HTML (ex. uma div). Tradução bruta seria Modelo de Caixa, e que leva a comparação que cada elemento HTML seria como uma caixa,

As 5 principais propriedades dele:

O Box Model consiste nessas 5 propriedades que permitem dimensionamento do elemento:

  • Width (largura)
  • Height (altura)
  • Padding (prenchimento)
  • Border (borda)
  • Margin (margem)

Para facilitar vamos imaginar que cada elemento HTML é uma pintura em um quadro e que ele está exposto em uma galeria de arte.
Começamos pegando um quadro separadamente.

Observe que o quadro possui varias partes que podem ser separadas, vamos pegar 3 delas por enquanto. A Pintura no centro, a Tela onde a pintura está, a Moldura .

Cada parte dessas seria como uma das propriedades do Box Model.

  • A Pintura seria o conteudo do elemento, texto, imagens, outros elementos, e como toda pintura possui uma altura por exemplo, essa possui 50cm de altura e 40 de largura, o nosso elemento se comportaria da mesma forma mas você pode definir o seu tamanho, pelas as propriedades Width e Height .
    ex:
    width: 400px,
    height: 300px
  • A Tela é a parte entre a Moldura e a Pintura, região branca, serve para destacar a pintura e deixa lá em um posiconamento melhor, ela seria o Padding, é o espaçamento do conteudo interno com a borda do elmento HTML, Ela não precisa estar centralizada, caso queira ela mais para a esquerda basta diminuir o padding apenas de um lado, veremos isso mais adiante.
  • A Moldura seria a Border do elemento, uma moldura pode possuir varios tipo, tamanhos e cores distintos. Da mesma forma que o quadro possui a moldura o elemnto HTML possui o border que é como se fosse uma moldura, pode colocar cores tamanhos e estilos diferentes, da forma que preferir.

Para o propriedade da margin precisamos analisar a imagem a abaixo.

  • Em uma galeria de arte nenhum quadro fica muito proximo ao outro, eles possuem uma distância entre eles, como a imagem mostra. A Margin é esse espaço a ser respeitado entre os quadros, a distância entre os elementos da sua pagina, para não ficarem colados, se destacarem e organizados.

    • Adicionar valores as propriedades

    Como definir os valores para eles.

    • Para o height e width é simples a forma de adicionar os valores, cada um possui apenas um valor.
      Exemplo:

      .div-color: {
      width: 200px;
      height: 100px;
      }

      Sendo que os valores ápos os numeros nem sempre precisam ser na medida px(pixel), pode usar outras como porcentagem do elemento pai(%),vw que é viewPort width(largura) a porcentagem da tela do browser do usuario, vh igual ao vw mas para height(altura),

    • As outras propriedades podem ser declaradas da mesma forma que width e height, mas todos os lados possuiram o mesmo valor.(como na div da dica.).
      Você pode declara apenas para um lado um valor, por exemplo:
      Verde
      Nessa div os padding estão com um valor unico.

      .div-verde: {
      background: green;
      border: 2px solid black;
      font-size: 30px;
      margin: 5px;
      padding: 20px;
      width: 80px;
      }

      Olhe como o padding pode ser declarado tambem.
      padding: 20 20 20 20px
      Nada muda, podemos mudar assim também,

      Verde

      .div-verde: {
      background: green;
      border: 2px solid black;
      font-size: 30px;
      margin: 5px;
      padding:0 20 20 20px;
      width: 80px;
      }

      Modificando dessa forma podemos ver que o padding superior ficou zerado ou - (pode inspecionar o elemento), e com isso o conteudo(Verde) está quase colado no topo do elemento.
      Quando colocamos esses quatro valores (0 20 20 20), o 0 seria o topo e continuaria em ordem horaria.
      top->right->bottom->left (topo->direita->baixo->esquerda)
    • Cada lado possui um nome og, queira mudar só a margin superior, você precisa colocar margin-top.
    • A Border possui outros valores a serem adicionados, estilo e cor. Pode ver nos exemplos superiores em como a border ficou declarada.
      Exemplo:
      border: 3px dotted green;

    Cuidado!

    Lembre sempre que o tamanho total do seu elemento sempre será:
    Largura Total = width + padding-left + padding-right +border-left + border-right.
    Mesma coisa para a Altura apenas trocar right e left por top e bottom e width por height.
    Para resolver esse problema pode usar a propriedade box-sizing: border-box.
