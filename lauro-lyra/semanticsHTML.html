O problema da tag <div>

Ao elaborar suas primeiras páginas em HTML, muito provaelmente você se utilizou das tags div para dividir as muitas seções de seu arquivo, bem como a tag p para designar o texto atribuído. Algo similar como a estrutura da imagem abaixo:

Note, todavia, que a elaboração de um site com todas as marcações iguais pode causar muitas dores de cabeça em vários sentidos. Em primeiro lugar, caso um outro desenvolvedor (diferente de quem criou a página) precise fazer alterações na página, certamente será gasto muito tempo somente para ler o arquivo HTML e compreender, ao final, qual parte se refere ao menu principal, qual dos trechos se trata de um artigo, qual deles aborda uma seção, qual das divs está o rodapé (isso se existe um!) e por aí vai.

Fazendo uma breve comparação, imagine que você tem que pagar um boleto num banco e, ao chegar lá, você se depara com várias portas, todas elas escritas 'sessão'. Como você saberá, olhando essas portas, qual das portas te levará até o caixa? Você passará muito tempo abrindo várias portas e inspecionando para, finalmente, descobrir qual das salas é a que te interessa.

E o problema de organizar tudo em divs não para por aí! Uma página web com marcações que não são intuitivas geram enormes problemas para softwares de acessibilidade e leitura de tela, que, assim como nós, humanos, não sabemos a que seção cada texto se refere quando ele está envolto numa mera tagdiv. Da mesma forma, mecanismos de pesquisa (como o google) terão dificuldade de indexar sua página porque também não sabem qual é o conteúdo priuncipal, menu e etc.

Pensando em resolver esses vários problemas, a mais nova implementação do HTML trouxe, como novidade, as chamadas tags semânticas, que auxiliam usuários e desenvolvedores a uma melhor compreensão de cada uma das marcações.

HTML semântico: conceitos

Semântica, por definição, quer dizer "o estudo do significado das palavras". Aplicado isso à linguagem HTML, temos que HTML semântico consiste na aplicação de tags com nomes que têm um significado.
