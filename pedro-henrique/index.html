Seletores de elementos do DOM com JavaScript

Parte 1: Teoria

O que é DOM?

Antes de aprender a selecionar os elementos de nossa página é importante entender um pouco do conceito de DOM (Document Object Model).O DOM é uma forma de interpretar uma página HTML de modo que fique orientada a objetos, podendo assim ser manipulada por alguma linguagem script. Esta linguagem script, como o javascript, consegue acessar através de linhas de comando cada elemento da página, podendo selecioná-los, acessar propriedades e até alteará-los.

Na imagem abaixo, podemos ver um exemplo da estrutura hierárquica de uma página HTML. Cada uma das 'Tags' é um elemento acessível pela linguagem script.

O conceito de DOM parece ser algo complexo, porém, com o tempo, este ficará bem mais claro em sua cabeça. Na próxima sessão aprenderemos a selecionar os elementos da paǵina HTML e dessa forma começaremos a perceber o poder do DOM!

Seletores de elementos

Nesta sessão, aprenderemos sobre os principais seletores da linguagem JavaScript. Para cada seletor, faremos exercícios de fixação. Os exercícios de fixação deverão ser realizados no console do próprio navegador, e serão baseados nesta página.

Vamoooos?

getElementsByTagName

O primeiro seletor que aprenderemos será o getElementsByTagName. Sua sintaxe é document.getElementsByTagName('nomeDaTag'). O retorno será um HTMLCollection.

HTMLCollection é uma coleção de nós, cada um contendo uma das tags do documento. Cada um desses nós podem ser acessados por meio de índices.

A principal propriedade do HTMLCollection é length.

Exercícios de fixação
  1. Quantas tags são 'h1', 'p' e 'div', respectivamente?
  2. Selecione as todas as tags 'h2'. O que é imprimido no console?
  3. Acesse o segundo 'h3'. Qual o texto dele? Consegue alterá-lo para "Círculos!"? Dica: utilize a propriedade '.innerHTML'

getElementsByClassName

O segundo seletor que aprenderemos será o getElementsByClassName. Sua sintaxe é document.getElementsByClassName('nomeDaClasse'). O retorno também será um HTMLCollection.

Exercícios de fixação
  1. Quantos elementos possuem as classes "verde" e "vermelho", respectivamente?
  2. Selecione a classe "azul". O que é imprimido no console?
  3. Acesse o terceiro elemento da classe "azul". Qual a cor do fundo dele? Consegue alterá-la para vermelho? Dica: utilize a propriedade '.style'

getElementById

O terceiro seletor que aprenderemos será o getElementById. Sua sintaxe é document.getElementById('nomeDoId'). O retorno não será um HTMLCollection. Por quê? Ora, se o id é individual do elemento, não há porque este seletor retornar diversos elementos!

Exercícios de fixação
  1. Selecione o elemento que possua o id "primeiro-circulo". Qual a sua cor? Consegue mudá-la para amarelo?.
  2. Acesse o elemento de id "segundo-circulo". O que está escrito nele? Consegue deixar a letra maior?

querySelector e querySelectorAll

O querySelector é um seletor mais versátil que aqueles que já aprendemos. Ele aceita qualquer uma das três formas que já vimos para selecionar (id, classe e tag), podendo combinar ainda estas formas (veremos isso em um exercício). A desvantagem de se usar o querySelector está no fato de ele só retornar o primeiro elemento encontrado. Dessa forma, ele pode ser usado bem quando se tem apenas um elemento com certa tag ou classe, ou para selecionar algo pelo id, ou então quando temos dentro de uma tag apenas um elemento com a classe que se quer selecionar.

Sua sintaxe é document.querySelector('nome'). Para classes colocamos '.nomeDaClasse', para id colocamos '#id' e para tag a escrita é igual

Já o querySelectorAll retorna todos os elementos de acordo com o nome buscado. Sua sintaxe é document.querySelectorAll('nome') e seu retorno é uma NodeList.

Exercícios de fixação
  1. Faça os exercícios 1 e 2 da parte de getElementById utilizando querySelector
  2. Selecione o elemento que possua a classe 'azul' dentro da tag 'h3', usando querySelector. Dica: h3.azul
  3. Selecione o primeiro elemento que possua a tag 'h1' com o comando querySelector e altere seu texto.
  4. Faça os exercícios 1, 2 e 3 da parte de getElementsByTagName utilizando querySelectorAll
