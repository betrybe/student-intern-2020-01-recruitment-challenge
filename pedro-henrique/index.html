O que é DOM?

Antes de aprender a selecionar os elementos de nossa página é importante entender um pouco do conceito de DOM (Document Object Model).O DOM é uma forma de interpretar uma página HTML de modo que fique orientada a objetos, podendo assim ser manipulada por alguma linguagem script. Esta linguagem script, como o javascript, consegue acessar através de linhas de comando cada elemento da página, podendo selecioná-los, acessar propriedades e até alteará-los.

Na imagem abaixo, podemos ver um exemplo da estrutura hierárquica de uma página HTML. Cada uma das 'Tags' é um elemento acessível pela linguagem script.

O conceito de DOM parece ser algo complexo, porém, com o tempo, este ficará bem mais claro em sua cabeça. Na próxima sessão aprenderemos a selecionar os elementos da paǵina HTML e dessa forma começaremos a perceber o poder do DOM!
